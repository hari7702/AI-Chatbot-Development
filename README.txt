
# Financial Chatbot Prototype

## Functionality
This Python script implements a simple rule-based chatbot that answers predefined financial queries based on the financial data provided. The chatbot can answer the following queries:

1. "What is the total revenue for Company A in Year 2?"
2. "How has net income changed for Company B from Year 1 to Year 2?"
3. "What is the total revenue for Company X in Year 1?"

## Limitations
- The chatbot is limited to the predefined queries listed above.
- It does not process or understand queries outside of these predefined options.
- It cannot handle variations in natural language input.

## Usage
To use the chatbot, simply run the `financial_chatbot.py` script and input one of the predefined queries when prompted.
